网站的设计思路：

1.首先我选用京剧为主题。京剧是我国国粹文化，通过建设京剧国粹传统文化网站，能向大家展示京剧的优美之处。

2.通过向来访者展现目前的京剧内容，京剧文化，京剧大师等，来吸引来访者对京剧的喜爱。

3.同时还可以实时更新最新的京剧内容，京剧唱腔，京剧文化等。

4.和其他大型网站的结构类似。我设计了主页、新闻页、详情页、介绍页、作者内容页，等内容。
