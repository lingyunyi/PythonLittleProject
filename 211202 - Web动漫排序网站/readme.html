设计思路：该页面的设计思想是我从日常生活中观看的动漫网站（B站）思考而来，我想通过编写一个属于我自己的动漫排序网站，向用户展示目前最热的动漫榜单以及热点动漫内容，从而吸引用户观看我所推荐的动漫。

网站特色：本站的主题内容为动漫推荐，向用户推荐本月最热的动漫，有热点日漫，也有国产动漫。
